# 🪒 Pirayesh — ربات تلگرام آرایشگاه مردانه

ربات هوشمند نوبت‌گیری و مدیریت آرایشگاه با پشتیبانی از پرداخت کارت‌به‌کارت، تاریخ شمسی، مدیریت تعطیلات و پنل مدیریت کامل.

---

## ✨ امکانات

### 👤 کاربر
- ثبت‌نام با نام، نام خانوادگی و شماره تماس
- 📅 نوبت‌گیری از ۷ روز آینده (روزهای تعطیل نمایش داده نمی‌شوند)
- تاریخ شمسی با نام روز هفته در همه مراحل
- 💳 پرداخت کارت‌به‌کارت با ارسال رسید
- 💵 پرداخت حضوری
- 📋 مشاهده و لغو نوبت‌های فعال
- 📞 مشاهده اطلاعات آرایشگاه

### ⚙️ ادمین
- 📋 نوبت‌های امروز و همه نوبت‌ها با جزئیات کامل (نام، تلفن، تاریخ شمسی)
- ✅ تایید / رد / لغو / انجام‌شدن نوبت
- 💳 نوتیف واریزی حرفه‌ای (بدون ارسال خودکار عکس) + دکمه «مشاهده رسید»
- ✂️ مدیریت خدمات (افزودن، فعال/غیرفعال)
- 🕐 مدیریت زمان‌بندی (تعریف روز و ساعت‌های کاری)
- 🎌 مدیریت روزهای تعطیل (افزودن/حذف)
- 🏪 ویرایش اطلاعات آرایشگاه از داخل ربات
- 🔧 toggle فعال/غیرفعال کردن بخش‌ها
- 📊 آمار کلی

| بخش | توضیح |

## 🚀 نصب روی VPS

### پیش‌نیاز
- Ubuntu 20.04 / 22.04
- دسترسی root یا sudo
- توکن ربات از [@BotFather](https://t.me/BotFather)
- آیدی عددی ادمین از [@userinfobot](https://t.me/userinfobot)

### نصب یک‌مرحله‌ای

```bash
git clone https://github.com/moha100h/Pirayesh.git
cd Pirayesh
chmod +x install.sh
./install.sh
```

اسکریپت فقط دو چیز می‌پرسد:

```
Bot Token (from @BotFather): 
Admin Numeric ID (from @userinfobot): 
```

> پوشه `data/` به‌صورت خودکار ساخته می‌شود و دیتابیس SQLite داخل آن ذخیره می‌شود.

بقیه تنظیمات را از **پنل مدیریت ربات** انجام دهید.

│   │   ├── models.py      # User, Service, TimeSlot, Booking, Payment, HolidayDate
│   │   └── database.py    # اتصال async به SQLite
│   ├── services/
│   │   └── jalali.py      # تاریخ شمسی + نام روز هفته

├── docker-compose.yml

## 🔄 فلو کاربر

```
/start
  └─► ثبت‌نام (نام + نام خانوادگی + شماره تماس)
        └─► منوی اصلی
              ├─► 📅 نوبت‌گیری
              │     └─► انتخاب خدمت → تاریخ شمسی (۷ روز) → ساعت → تایید
              │           └─► پرداخت کارت / حضوری
              │                 └─► نوتیف ادمین (متنی + دکمه)
              ├─► ✂️ خدمات و قیمت‌ها
              ├─► 📋 نوبت‌های من
              └─► 📞 اطلاعات آرایشگاه
```
